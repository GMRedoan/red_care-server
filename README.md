# ❤️ Red Care Server

## About

Red Care Server is the backend API for the **Red Care** platform. It is built with **Node.js**, **Express.js**, and **MongoDB** to securely manage user information, donation requests, funds, and other application data. The server follows RESTful API principles and is designed to provide reliable, scalable, and efficient data management.

## 🚀 Getting Started

### Clone the repository

```bash
git clone https://github.com/GMRedoan/red_care-server.git
```

### Install dependencies

```bash
npm install
```

### Configure environment variables

Create a `.env` file and add the following:

```env
PORT=5000
MONGODB_URI=your_mongodb_connection_string
```

### Start the server

```bash
npm run start
```

or (for development)

```bash
npm run dev
```
